# Stockpile Web Game

A web-based multiplayer implementation of the Stockpile board game. 

## Features
- **Phase 1: Core Gameplay:** 2-5 player authoritative real-time game engine.
- **Phase 2-4:** See roadmap in implementation plan.

## Setup and Running

### Backend Server
Runs on port 2648, listening on all interfaces.
```bash
cd server
npm install
npm run dev
```

### Frontend Client
Runs on port 8462, listening on all interfaces.
```bash
cd client
npm install
npm run dev
```

## Technologies
- **Frontend:** React, Vite, TypeScript, Socket.io-client, CSS (Retro 80s Wall Street aesthetic)
- **Backend:** Node.js, Express, Socket.io, TypeScript
